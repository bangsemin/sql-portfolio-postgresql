# SQL Portfolio with PostgreSQL

이 저장소는 SQLD 시험 대비와 실무형 SQL 분석 역량 강화를 위해 만든 PostgreSQL 기반 SQL 포트폴리오입니다.

## 목표

- SQLD 핵심 개념 복습
- PostgreSQL 기반 실무 SQL 연습
- 데이터 모델링, JOIN, GROUP BY, 서브쿼리, 윈도우 함수 실습
- 분석 쿼리 작성 후 인사이트 도출
- GitHub 커밋 이력을 통한 학습 과정 기록

## 폴더 구조

- `00_setup`: 데이터베이스, 스키마, 샘플 데이터 생성
- `01_sqld_practice`: SQLD 개념별 SQL 연습
- `02_projects`: 실무형 SQL 분석 프로젝트
- `03_notes`: 개념 정리, 오류 기록, 복습 노트
- `scripts`: 실행 및 커밋 자동화 스크립트

## 학습 원칙

pgAdmin에서만 쿼리를 실행하지 않고, 모든 SQL 작업은 `.sql` 파일로 저장합니다.

저장된 SQL 파일과 분석 노트는 GitHub에 커밋하여 포트폴리오 이력으로 남깁니다.
